Para criar o arquivo Jar deste projeto: "springboot-rest-api-sample", no curso: CRUD com Spring Boot REST API, do professor: Alex Egídio, faça no sistema operacional Windows:

* Abra o Prompt de Comandos do Windows.
* Execute o comando para chegar a este diretório. "cd <caminho deste diretório>"
* Rode o comando: "mvn clean package". Obs.: Este comando requer que o Maven esteja "instalado nesta máquina" e devidamente configurado para
 ser executado via linha de comando.

E então, o arquivo resultante já está disponível neste mesmo diretório! ;)
Ele se chama:

"springboot-rest-api-sample-1.0.0-SNAPSHOT.jar"

.

E para executá-lo, rode o comando:

"java -jar springboot-rest-api-sample-1.0.0-SNAPSHOT.jar"

.
